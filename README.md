# GBR Security Bot

Telegram-бот для преобразования адресов в ссылки на Яндекс.Карты с использованием DaData API.

## Функционал

- **Команда /start**: Приветственное сообщение с инструкциями
- **Обработка адресов**: Преобразование текстового адреса в координаты
- **Интеграция с DaData**: Геокодирование адресов через DaData API
- **Ссылки на Яндекс.Карты**: Формирование прямых ссылок на карты

## Установка и настройка

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd GBR-Security-Bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

1. Скопируйте файл `.env.example` в `.env`:
```bash
cp .env.example .env
```

2. Получите токены:
   - **Telegram Bot Token**: Создайте бота у [@BotFather](https://t.me/BotFather) в Telegram
   - **DaData API Key**: Зарегистрируйтесь на [dadata.ru](https://dadata.ru/) и получите API ключ

3. Отредактируйте файл `.env`:
```env
TELEGRAM_BOT_TOKEN=your_actual_telegram_bot_token
DADATA_API_KEY=your_actual_dadata_api_key
```

## Запуск

### Запуск бота
```bash
python bot.py
```

Бот начнет работать и будет отвечать на сообщения в Telegram.

## Использование

1. Отправьте боту команду `/start` для приветствия
2. Напишите любой адрес в чат (например: "Ленина 5" или "Москва, Тверская улица, 1")
3. Бот найдет координаты и пришлет ссылку на Яндекс.Карты

## Примеры использования

```
/start
Ленина 5
Москва, Тверская улица, 1
Санкт-Петербург, Невский проспект, 50
```
